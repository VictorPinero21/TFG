Si quieres arrancar el proyecto tienes que hacer las siguientes cosas 
1.Crear un archivo llamado ".env" y poner todo lo que hay al final de este documento 
2.Abrir un terminal en la carpeta del proyecto y hacer un npm i para el nodemodules 
3.Hacer un composer install para las carpetas del vendor 
4.Ejecutar un php artisan serve en otro terminal diferente al del paso 2
